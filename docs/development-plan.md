## Ключевые архитектурные решения

### Расчёт TSS
- Для мощности: `TSS = (duration_sec × NP × IF) / (FTP × 3600) × 100`
- Для ЧСС (без мощности): hrTSS через зоны ЧСС
- Для бега: rTSS через нормализованный темп

### CTL / ATL / TSB
```
CTL(today) = CTL(yesterday) × e^(-1/42) + TSS(today) × (1 - e^(-1/42))
ATL(today) = ATL(yesterday) × e^(-1/7)  + TSS(today) × (1 - e^(-1/7))
TSB(today) = CTL(yesterday) - ATL(yesterday)
```

### Контекст ИИ-агента (структура промпта)
```
System: Ты эксперт по планированию тренировок для циклических видов спорта и OCR...
        Принципы периодизации...
        Правила нагрузки...

User context:
  - Профиль спортсмена
  - Текущая фаза (base/build/peak/race/recovery)
  - CTL/ATL/TSB сегодня
  - История тренировок (последние 8 недель, по неделям)
  - Тренд HRV и Recovery Score
  - Целевые гонки
  - Запрос пользователя

Request: Составь план на следующие 2 недели...
```
